Chalk npm library

NPM Chalk library in nodeJs is used for styling the format of text and it allows us to customize our own themes in our nodeJs projects. We can customize the color of the command_line output using Chalk.

importing Chalk :

Step1 : import the library

const chalk = require('chalk'); 



            Step 2: using it within our code.
        


        
 Using Chalk :


        

            console.log(chalk.color_name("Hello,world!"));
            

            We can also add more styles to our customized text, like another example is shown below.
            

            console.log(chalk.bold.color_name("Hello,world!"));
